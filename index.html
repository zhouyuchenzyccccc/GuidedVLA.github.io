Specifying Task-Relevant Factors via Plug-and-Play Action Attention Specialization

Vision-Language-Action models often develop shortcut-prone representations that conflate task semantics with incidental visual statistics. GuidedVLA transforms the action decoder into a robust, interpretable system through functional attention head specialization.

Method: Functional Attention Head Specialization

Instead of allowing cross-attention heads to develop roles implicitly, GuidedVLA manually specifies the information each head should capture by supervising them with distinct, task-relevant auxiliary signals. This structural constraint enforces functional decoupling of the action decoder, ensuring critical factors are mapped to dedicated semantic subspaces.

Object Grounding Head

Supervises attention maps to explicitly ground task-relevant objects and suppress distractors via attention mask alignment loss. Critical for precise localization on transparent/refractive objects and small targets.

Key insight: Forces action tokens to attend to semantically meaningful regions rather than incidental visual contrast.

Skill Recognition Head

Aligns internal feature representations with temporal skill phases (e.g., Pick → Place) through auxiliary classification loss. Prevents stage-skipping in multi-step behaviors.

Key insight: Encodes temporal intent progression to maintain stage awareness across extended horizons.

Geometry Perception Head

Injects explicit 3D spatial information by constraining dedicated attention heads to process only features from a frozen depth encoder (Depth Anything 3).

Key insight: Provides metric geometric reasoning for sub-centimeter precision tasks where monocular RGB cues are insufficient.

Experimental Results

GuidedVLA achieves significant performance gains across simulation benchmarks and real-world platforms, with particularly strong improvements under distribution shifts.

Robotwin 2.0 Benchmark

Average success rate improved from 77.38% (π₀ baseline) to 90.63% with all three specialized heads.

Single-head ablations reveal task-specific alignment: depth head excels on geometry-heavy tasks (Beat Block Hammer: 78% → 99%), skill head dominates long-horizon tasks (Lift Pot: 92% → 99%).

Factor Quality Correlation

Higher factor quality directly correlates with improved task performance:

  • Object grounding: Attention mass on target ↑ from 0.25 → 1.0 → Success ↑ 61.3% → 74.6%
  • Skill recognition: Probe accuracy ↑ 0.25 → 1.0 → Success ↑ 66.2% → 72.9%
  • Geometry perception: Depth feature ratio ↑ 0 → 1.0 → Success ↑ 15.6% → 76.7%

Failure Case Analysis Across Domains

Failures across all six tasks share a unified origin: end-to-end training without explicit factor guidance causes the action decoder to develop shortcut-prone representations that conflate task semantics with incidental visual statistics. Critically, these limitations become safety-critical in chemically complex environments.

Household Tasks (1-3)

ALOHA Platform Failures

Task 1: "Phantom grasping" on small strawberry due to reliance on lighting-dependent contrast rather than invariant identity.

Task 2: "Half-grasp" failure in bowl stacking from insufficient penetration depth estimation (<5mm clearance).

Task 3: Stage-skipping behavior—completes pouring but skips mandatory tool return phase.

ChemLab Tasks (4-6)

PSI-Bot Platform Failures

Task 4: Phantom grasp on transparent beaker due to refraction-induced appearance instability; mantle rim collisions from clearance misjudgment.

Task 5: Lighting-induced miss-grasps on small beaker; beaker-beaker collisions during placement due to insufficient metric clearance control.

Task 6: Premature release of wire gauze before stabilization causes roll-off; skewed beaker placement leading to instability.

Cross-Domain Failure Amplification

The same fundamental limitations observed in household settings become safety-critical in laboratory environments:

  • Object grounding failures: Phantom grasps on transparent glassware risk equipment damage and chemical spills
  • Geometric reasoning failures: Sub-millimeter clearance errors cause collisions that are merely task failures at home but safety hazards in chemical environments
  • Temporal skill failures: Premature release of lab equipment causes breakage/spillage with potential chemical exposure risks

GuidedVLA addresses these limitations through functional decoupling—transforming the action decoder into a robust, interpretable system where failures become diagnosable and addressable through targeted supervision.
